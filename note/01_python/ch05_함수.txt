3절. 함수의 매개변수
        -매개변수에 기본값을 가질 때 : 호출시 선택적으로 명시
            - 매개변수 기본값이 스칼라 변수일 때 : 기본값이 함수 정의 시점
            - 매개변수 기본값이 객체일 때 : 기본값이 함수 호출 시점
        - 키워드 인수 : 기본값 갖고 호출시 매개변수 명을 반드시 기입해야 하는 경우
        - 튜플 매개변수 : 함수 선언시 *args
        - 딕셔너리 매개변수 : 함수 선언시 **kargs 
        - 튜플 인수 언패킹 :  함수 호출시 *args
        - 딕셔너리 인수 언패킹 : 함수 호출시 **kargs  (args, kargs는 변수)

    *함수 정의시 매개변수 순서*
        순서인수 > 튜플인수 > 키워드인수 > 딕셔너리인수
