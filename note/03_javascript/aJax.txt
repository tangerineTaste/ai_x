1장 선택자로 불러오기
    $(선택자).함수 : 선택자를 불러와 함수를 실행할 수 있음

2장 선택자 함수
    $(선택자).val() : 선택자의 value 를 return 혹은 ()에 값을 입력할 수 있음
    $(선택자).text() : 선택자의 text ``
    $(선택자).html() : 선택자의 hmtl `` . text와 다른 점은 태그가 적용됨
    $(선택자).css() : ()안에 css를 입력할 수 있다.

3장 이벤트
    $(선택자).이벤트(콜백함수) : 이벤트 발생 시 콜백함수를 비동기적으로 실행 할 수 있다.

4장 ajax 함수 : $.get(), $.post(), $.ajax()
    $.get(요청경로,callback(data,status){})
    위 방식처럼 사용하여 submit에 대한 처리를 할 수 있다.
    data : 요청경로를 수행한 결과
    status : success / fail
